# Vow — Capacitor Native App Setup

Turn your single-file `Vow — Retention Tracker v3.html` into a native iOS and Android app using [Capacitor](https://capacitorjs.com/).

---

## Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| Node.js | ≥ 18 | https://nodejs.org |
| npm | ≥ 9 | bundled with Node |
| Xcode | ≥ 15 | Mac App Store (iOS only) |
| Android Studio | ≥ 2023 | https://developer.android.com/studio |
| CocoaPods | latest | `sudo gem install cocoapods` (iOS only) |

---

## Project layout

```
vow/
├── Vow — Retention Tracker v3.html   ← your existing single-file app
├── package.json
├── capacitor.config.json
├── scripts/
│   └── build.js                      ← copies HTML → www/index.html
└── www/                              ← generated by build script (git-ignored)
    ├── index.html
    └── manifest.webmanifest
```

---

## Step 1 — Install dependencies

```bash
npm install
```

---

## Step 2 — Initialise Capacitor (first time only)

```bash
npx cap init "Vow" "com.yourcompany.vow" --web-dir www
```

---

## Step 3 — Add platforms (first time only)

```bash
npx cap add ios
npx cap add android
```

---

## Step 4 — Build web assets

```bash
npm run build
```

---

## Step 5 — Sync to native projects

```bash
npm run sync      # runs build + cap sync
```

---

## Step 6 — Open native IDEs

```bash
npm run ios       # opens Xcode
npm run android   # opens Android Studio
```

---

## Step 7 — App icons & splash screens

```bash
npm install --save-dev @capacitor/assets
npx capacitor-assets generate --iconBackgroundColor '#1A1A1C' --splashBackgroundColor '#1A1A1C'
```

Place source files at:
```
assets/
├── icon-only.png        (1024×1024)
├── icon-foreground.png  (1024×1024, Android adaptive)
├── icon-background.png  (1024×1024, solid #1A1A1C)
└── splash.png           (2732×2732)
```

Full guide: https://capacitorjs.com/docs/guides/splash-screens-and-icons

---

## Step 8 — Build for release

### iOS
1. `npx cap open ios`
2. Select your team in **Signing & Capabilities**
3. Set Bundle ID to `com.yourcompany.vow`
4. **Product → Archive** → distribute via App Store Connect

### Android
1. `npx cap open android`
2. **Build → Generate Signed Bundle / APK**
3. Upload `.aab` to Google Play Console

## Changes needed in index.html

### Notifications (iOS)
The browser `Notification` API doesn't work in Capacitor on iOS. Add these helpers:

```js
async function requestNativeNotifPermission() {
  if (!window.Capacitor) return Notification.requestPermission();
  const { LocalNotifications } = Capacitor.Plugins;
  const { display } = await LocalNotifications.checkPermissions();
  if (display !== 'granted') {
    const result = await LocalNotifications.requestPermissions();
    return result.display === 'granted' ? 'granted' : 'denied';
  }
  return 'granted';
}

async function sendNativeNotif(title, body) {
  if (!window.Capacitor) { new Notification(title, { body }); return; }
  const { LocalNotifications } = Capacitor.Plugins;
  await LocalNotifications.schedule({
    notifications: [{ title, body, id: Date.now(),
      schedule: { at: new Date(Date.now() + 500) },
      iconColor: '#3B8E5E' }]
  });
}
```

Replace `Notification.requestPermission()` → `requestNativeNotifPermission()`
Replace `new Notification(...)` → `sendNativeNotif(...)`

### External links
```js
document.addEventListener('click', e => {
  const a = e.target.closest('a[href]');
  if (a && a.href.startsWith('http') && !a.href.includes(window.location.host)) {
    e.preventDefault();
    if (window.Capacitor) Capacitor.Plugins.Browser?.open({ url: a.href });
    else window.open(a.href, '_blank');
  }
});
```
